## Dev Notes

**Running the Project**

*note: clone this project first on your machine*
1. On your terminal, do > `cd Decimal32FP-Sim`
2. also do > `npm install`
3. then to run app on dev server > `npm run dev`  
4. You should see app running on localhost:3000
